# Boost Utils

## Objective

A collection of utility functions and more things that can be useful in future projects.

## What's inside?

[logger](/logger.js) - A simple logger utility to log messages with some pattern.

[db-helpers](/db-helpers.js) - A set of database helper functions to manage transactions and connections.

[milestones](/milestones.md) - A overview of milestones created in `curso.dev` to track the progress of the project.
